# Nserver — Painel Pessoal Desktop

MVP local para Windows.

## Como usar no notebook Windows

1. Abra a pasta `nserver`.
2. Dê dois cliques em `iniciar-nserver.bat`.
3. O terminal vai mostrar os endereços:
   - No notebook: `http://127.0.0.1:8791`
   - No celular: `http://IP-DO-NOTEBOOK:8791`
4. No celular, esteja na mesma rede Wi-Fi do notebook e abra o endereço informado.

## Login inicial

Use as credenciais combinadas no chat.

## O que já existe neste MVP

- Tela de login.
- Tela de boas-vindas.
- Data/hora atual.
- Dashboard principal.
- Estrutura inicial para ferramentas futuras.

## Observações

- Este MVP não instala integrações externas.
- Ele usa apenas Python padrão, sem dependências extras.
- Se o Windows Firewall perguntar, permita acesso na rede privada.
